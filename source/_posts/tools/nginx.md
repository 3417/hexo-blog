## 使用

1. 执行文件夹中的nginx.exe
2. 进入该文件中执行start nginx

## 常用命令

```
1. 启动服务 start nginx
2. 退出服务 nginx -s quit
3. 强制关闭服务 nginx -s stop
4. 重载服务 nginx -s reload
5. 验证配置文件 nginx -t
6. 使用配置文件 nginx -c '配置文件路径'
7. 使用帮助 nginx -h
```
