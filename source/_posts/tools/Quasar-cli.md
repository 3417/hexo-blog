## 安装+使用

{% codeblock %}
yarn global add quasar-cli OR npm install global quasar-cli  //old
yarn global add @quasar/cli OR npm global add @quasar/cli //new

quasar init <文件夹名称>  //old
quasar create <folder_name> //new
{% endcodeblock %}
